To truly test the allowlist, open this page on port 9000 (http://localhost:9000/test-widget.html) — that's cross-origin from the chat API on port 8000, so CORS will kick in. Opening it on port 8000 is same-origin and bypasses CORS.

<script src="http://localhost:8000/widget/loader.js" data-brand="rcuk"></script>
The widget bubble should only appear if this page's origin is listed in Admin Settings → Allowed Widget Domains.
